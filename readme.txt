Index html como pagina principal, carregando via DOM e JS, as funcoes necessarias para editar, criar e deletar novas marcas e modelos de.
 Index: Pagina principal
 Botao Select com marca "Audi" pre-selecionada: Atua como seletor de marca dos veiculos
 Marca: Cada marca carrega sua tabela com modelos respectivos, tais como: Modelo, Cor e Motor.

 Botao Editar: Abre prompt para edição de nome modelo, cor e motor do respectivo modelo.
 Botao Excluir: Exclue o respectivo elemento da linha do botal selecionado.
